MERN STACK PROJECT 2024

Packages to be installed:

1) Vite (Client side)
   npm install -g vite
   npm install vite --save-dev

2) nodemon (Server side)
   npm install -g nodemon
   npm install nodemon --save-dev


Execution steps:
1) Server side - npm start
2) Client side - npm run dev
